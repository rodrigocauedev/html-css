Principais formatações

Negrito/Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase, tenho um termo em destaque usando a tag <strong> (Semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <EM> (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E no outro parágrafo, temos outro texto marcado no final.

Para criar uma marcação se usa a TAG <mark>.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

A TAG para criar um texto grande é <big>, mas é uma TAG que pode sair do HTML5 a qualquer momento. A TAG para criar um texto pequeno é <small>.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele dever ser lido, mas não considerado,

A TAG para indicar um texto deletado é <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele não foi adicionaco depois. A TAG usada é <ins>.

Existe também o sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3, use a TAG <sup>.

Texto subscrito

Para inserir coisas do tipo H2O, use a TAG <sub>.
